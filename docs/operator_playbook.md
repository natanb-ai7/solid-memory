# Operator Playbook (10-minute daily workflow)

1. **Upload fresh data (2–3 min)**
   - Update portfolio positions, funding terms, hedge book, market indicators, and M&A CRM.
   - Use Manual Entry for quick single-record fixes.
2. **Run the memo (1 min)**
   - Click “Run Today” in the dashboard or `prewire run --date YYYY-MM-DD`.
3. **Review fix-it report (1–2 min)**
   - If validation fails, follow the suggested fixes and re-run.
4. **Scan the memo (3–4 min)**
   - Focus on Top 3 Decisions, Top 3 Risks, Liquidity readiness, and Action triggers.
5. **Export & share (1 min)**
   - Use PDF or JSON for downstream workflows.
   - Toggle redaction mode if counterparty names should be hidden.
