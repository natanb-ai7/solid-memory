# Program Monitor SPA

Single-page React application (Vite + React Query) for tracking current programs, historical events, and source freshness.

## Features
- Pages for **Current Programs**, **History**, and **Source status** with shared navigation.
- API layer backed by mock data with client-side caching via React Query.
- Sorting, filtering, and CSV export on the Current Programs grid.
- Data freshness badges for each make/region across pages.

## Getting started
```bash
npm install
npm run dev
```

Build for production with `npm run build`.
